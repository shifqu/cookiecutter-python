# cookiecutter-python

A simple template for my own personal Python projects utilizing pip-tools + pylint + ruff + pyright. Best used with [cruft](https://cruft.github.io/cruft/)

To use:

```
cruft create https://github.com/shifqu/cookiecutter-python/
```
